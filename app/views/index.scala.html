@(host:String) Jira to Sladiator
Jira to Sladiator

Description

This is a free site to be used as a proxy from Jira webhooks to the API of sladiator.

More information about Jira webhooks: https://developer.atlassian.com/display/JIRADEV/JIRA+Webhooks+Overview
More information about the sladiator API: https://sladiator.com/forums/tips-tricks/topics/public-api

Usage

Simply point your Jira webhook to: http://@host/webhook/{SLA Token}

An example: http://@host/webhook/4bx60a464xd23b5x2ad715cc1fc059be8cc1a9eb
